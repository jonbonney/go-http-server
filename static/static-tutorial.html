Serving Static Files with Go

Learn how to easily serve static files with a Go HTTP server.

Introduction

This tutorial will guide you through the steps to set up a basic HTTP server in Go that serves static files, such as HTML, CSS, and JavaScript files. Serving static files is a common requirement for many web applications.

Prerequisites

Before you begin, you should have the following installed on your computer:

  • Go (at least version 1.11 to use Go modules)

Step 1: Create Your Project

Create a new directory for your project and navigate into it:

mkdir go-static-server
cd go-static-server

Step 2: Initialize a Go Module

Initialize a new Go module:

go mod init go-static-server

Optionally, create a new Git repo and make your initial commit:

git init
git add .
git commit -m "Initial commit"

Step 3: Create the Static Files

Create a directory called 'static' and place your HTML, CSS, and JavaScript files inside this directory. For example, here's a simple HTML file:

<!DOCTYPE html>
<html>
<head>
    <title>Welcome to Go Static Server</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

Step 4: Write the Server Code

Create a file named main.go in your project directory and add the following code:

package main

import (
    "net/http"
)

func main() {
    // Set up file server handler
    fs := http.FileServer(http.Dir("static"))
    http.Handle("/", fs)

    // Start server
    println("Server starting on port 8080...")
    if err := http.ListenAndServe(":8080", nil); err != nil {
        println(err.Error())
    }
}

Step 5: Run Your Server

Run your server using the command:

go run main.go

Now, open your web browser and visit http://localhost:8080. You should see your static HTML page.

Conclusion

In this tutorial, you've learned how to set up a basic HTTP server in Go that serves static files. This setup is suitable for hosting simple websites or the static portions of a web application.
